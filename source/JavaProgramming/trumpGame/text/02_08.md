```
    }
```

### ②和名の属性を追加

```
    public enum SuitEnum {

        SPADE("スペード"),
        HEART("ハート"),
        DIA("ダイヤ"),
        CLUB("クラブ");

        String name;

        /* コンストラクタ */
        private SuitEnum(String name) {
            this.name = name;
        }

        /* アクセサ */
        public String getName() {
            return this.name;
        }

    }
```

### ③和名で出力

```
    SuitEnum suit = SuitEnum.SPADE;
    System.out.println(suit.getName());
```
