## 補足情報の記載方法

```{hint}
これは``hint``型の注釈です。
```

```{note}
これは``note``型の注釈です。
```

```{attention}
これは``attention``型の注釈です。
```

    注釈の種類として、attention（注意） / caution（注意） / danger（危険） / error（エラー） / hint（ヒント） / important（重要） / note（注釈） / seealso（参考） / tip（Tip） / warning（警告）がプリセットされています
